# Plotly.js Examples

Ein modulares Projekt zur Darstellung von Datenvisualisierungen mit Plotly.js. Entwickelt als Grundlage für ein HMI-System.

## Übersicht

Dieses Projekt demonstriert verschiedene Plotly.js-Visualisierungen in einer responsiven Grid-Layout-Struktur. Die Plot-Funktionen sind modular aufgebaut und können flexibel kombiniert werden.

## Features

- ✅ Modulare Plot-Struktur mit ES6 Modules
- ✅ Responsive Grid-Layout (Content + Menu)
- ✅ Dynamische Plot-Generierung
- ✅ Flexible Datenübergabe an Plot-Funktionen
- ✅ Vorbereitet für HMI Integration

## Verfügbare Plots

- **Vertical Bar Chart** - Vertikales Balkendiagramm
- **Horizontal Bar Chart** - Horizontales Balkendiagramm
- **Bubble Plot** - Bubblediagramm mit verschiedenen Symbolen


## Installation

1. Repository klonen:
```bash
git clone https://github.com/PollyMesser/plotlyJs-examples.git
cd plotlyJs-examples
```

2. HTML-Datei im Browser öffnen
